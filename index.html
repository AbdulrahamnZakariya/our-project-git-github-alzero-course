أسلوب مبتكر لتعلم Git و GitHub

دليل شامل للتعلم السريع

  • إنشاء حساب على GitHub: ابدأ رحلتك في البرمجة وانضم لمجتمع المطورين العالمي.
  • تعلم إنشاء المستودعات: نظّم مشاريعك البرمجية باستخدام git init.
  • أوامر التنقل: استخدم cd للوصول إلى المجلدات وls لعرض الملفات.
  • إنشاء الملفات والمجلدات: استخدم mkdir لإنشاء مجلد جديد وtouch لإنشاء ملف.
  • استنساخ المشاريع: حمل المستودعات باستخدام git clone.
  • تأمين الاتصال باستخدام SSH:
    1. إنشاء مفتاح SSH: ssh-keygen.
    2. نسخ المفتاح إلى حسابك: cat ~/.ssh/id_rsa.pub.
    3. اختبار الاتصال: ssh -T git@github.com.
  • التعامل مع الفروع (Branches):
    1. عرض الفروع: git branch.
    2. إنشاء فرع جديد: git checkout -b branch-name.
    3. دمج الفروع: git merge branch-name.
  • إدارة التعديلات:
    1. إضافة التغييرات: git add ..
    2. تسجيل التعديلات: git commit -m "رسالة التعديل".
    3. رفع التعديلات: git push origin branch-name.
  • فتح Pull Request: قدم طلبًا لمراجعة تغييراتك عبر واجهة GitHub.
